# Proyecto Semana 04 — Catálogo Interactivo (Dominio: Inmobiliaria)

Este es un proyecto ejemplo que contiene un catálogo interactivo para una inmobiliaria.

Estructura mínima incluida:

- `src/components`: componentes React en español (Catálogo, ItemCard, ItemList, SearchBar, FilterPanel, SortSelector, EmptyState, LoadingSpinner)
- `src/data/items.ts`: datos mock de propiedades
- `src/hooks/useDebounce.ts`: hook de debounce para la búsqueda
- `package.json`, `index.html` y `src/main.tsx` para ejecutar con Vite

Ejecutar:

```bash
cd "weekend 4"
npm install
npm run dev
```

Abre http://localhost:5173/ para ver la app.
