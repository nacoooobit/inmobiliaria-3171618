# Proyecto Semana 04 — Catálogo Interactivo (Dominio: Inmobiliaria)

Implementación en español de un catálogo interactivo para una inmobiliaria.

- Dominio: Inmobiliaria (lista de propiedades, filtros, búsqueda y ordenamiento)
- Estructura: `starter/` incluye el catálogo en `src/catalog`.

Instrucciones rápidas:

1. Abrir `week 3/3-proyecto/starter` y ejecutar tu entorno (por ejemplo con Vite)
2. `npm install` y `npm run dev`

Notas:

- Los datos mock están en `src/catalog/data/items.ts`.
- El catálogo demuestra renderizado condicional, filtros, orden y búsqueda con debounce.
