Proyecto Week 5 — Sistema de Configuración UI (Inmobiliaria)

Instrucciones rápidas:

1. Instalar dependencias:

```bash
npm install
```

2. Ejecutar en modo desarrollo (usa Vite):

```bash
npm run start
```

Descripción: carpeta `starter` contiene un `ConfigProvider`, compound components (`Card`, `Form`), `SettingsPanel` y una `App` de ejemplo que muestra propiedades inmobiliarias y aplica la configuración global.
